# Requisitos do Sistema - Vitrine de Artesanato

## 1. Requisitos Funcionais (RF)

* **RF01 - Listagem de Produtos:** O sistema deve exibir uma vitrine com foto, título, nome do artesão e preço de cada produto cadastrado.
* **RF02 - Contato via WhatsApp:** O sistema deve permitir que o cliente inicie uma conversa no WhatsApp diretamente com o vendedor ao clicar no botão "Comprar".
* **RF03 - Busca de Produtos:** O sistema deve permitir filtrar produtos pelo nome ou categoria (Item previsto no roadmap).
* **RF04 - Visualização Responsiva:** A interface deve se adaptar automaticamente a telas de computadores e dispositivos móveis.

## 2. Requisitos Não-Funcionais (RNF)

* **RNF01 - Usabilidade:** A interface deve ser simples e intuitiva, focada em usuários com pouca familiaridade tecnológica (validado com público-alvo).
* **RNF02 - Disponibilidade:** O sistema deve operar em arquitetura Web, acessível via navegador.
* **RNF03 - Desempenho:** As imagens devem ser otimizadas para carregamento rápido em redes móveis (3G/4G).

## 3. Regras de Negócio (RN)

* **RN01 - Venda Direta:** A plataforma não processa pagamentos; ela serve apenas como ponte de contato.
* **RN02 - Valorização Local:** O nome do artesão deve ter destaque igual ou superior ao nome do produto.
